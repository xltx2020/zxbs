数据标注

关键点 / Key point

通过人工标注目标关键点信息，获取目标特征，对目标属性或形态进行辨识，常用于骨骼点标记、人脸标注等。

线及轮廓 / Lane&Contour

标注线性目标或物体轮廓。常用于车道线标注（自动驾驶）、人体轮廓标注、医学影像标注、场景物体标注等。

2D框 / Bounding box

将目标用最小化矩形框标出，附加多重属性，常用于车辆标注、人脸标注、行人标注等区域检测、目标检测类型数据标注。

语义分割 / Segmentation

通过像素级别的标注，将图片各区域分割标注，是计算机可以深入理解图片的每个细节。可以应用于地理信息系统、无人驾驶领域、医疗影响分析、智能机器人等领域。

3D框 / 3D Bounding box

用三维立体框对图片中指定目标进行标注，用于从2D图像或视频中进行空间认知。常应用于无人驾驶、智能安防等领域。

点云&跟踪 / Point cloud

在雷达扫描生成的3D空间中，对目标进行激光点标注、区域标注或3d框标注，更可以在连续数据中进行点云跟踪标注。
